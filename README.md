# Facial_Recognition_Attendance_System

Aim and Objectives of this Project

The aim of this project is to design an automatic attendance system using face recognition.

- Setup Raspberry pi 4 as the main controller.
- Setup a camera to capture the faces of the students for registration and verification.
- Take head-shots of students and label them according to their ID numbers.
- Create a folder to act as database to store images of users already taken and labelled in step 2. This will act as registration and also for matching reference.
- Program the system using Python to manage the image recognition, verification and storage.
- Save verified students’ details, such as student’s ID number, date and time of class or exam attendance to a .csv file.
- Automatically go through the class attendance of each student for the entire semester and select students with over 75% attendance to qualify for the exams.
